Interface BitBadgesUserInfo

BitBadgesUserInfo is the type for accounts returned by the BitBadges API

Remarks

collected, activity, announcements, and reviews are profile information that is dynamically loaded as needed from the API. The pagination object holds the bookmark and hasMore information for each of collected, activity, announcements, and reviews.

For typical fetches, collected, activity, announcements, and reviews will be empty arrays and are to be loaded as needed (pagination will be set to hasMore == true).
