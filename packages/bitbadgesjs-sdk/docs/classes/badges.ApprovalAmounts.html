    Class ApprovalAmounts

    ApprovalAmounts defines approval amounts per unique "from," "to," and/or "initiated by" address. If any of these are nil or "0", we assume unlimited approvals. If they are set to a value, then the running tally of the amounts transferred for the specified token IDs and ownership times must not exceed the corresponding value.
