# Hosted OpenAPI Specification

This directory contains the final, processed OpenAPI specification for the BitBadges API, hosted for easy access by LLMs and other automated tools.

## Files

- `openapi.json` - OpenAPI specification in JSON format (recommended for LLMs)
- `openapi.yaml` - OpenAPI specification in YAML format

## Access URLs

Once committed to the repository, these files are accessible via:

- JSON: `https://raw.githubusercontent.com/BitBadges/bitbadgesjs/main/packages/bitbadgesjs-sdk/openapi-hosted/openapi.json`
- YAML: `https://raw.githubusercontent.com/BitBadges/bitbadgesjs/main/packages/bitbadgesjs-sdk/openapi-hosted/openapi.yaml`

## Auto-Generated

These files are automatically generated and updated by the CI/CD pipeline whenever the API documentation changes. Do not manually edit these files.
